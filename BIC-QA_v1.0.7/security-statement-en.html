Security Statement

Data Security Commitment: BIC places the highest priority on safeguarding user data security and privacy.

1. Data Privacy Protection

Your uploaded AWR report is used exclusively for this analysis task and is automatically deleted once the task is completed. BIC guarantees that it:

  • Does not retain any copies of the report;
  • Does not disclose, share, or sell user data to third parties;
  • Does not use the report for any purpose other than this analysis;
  • Strictly complies with applicable data protection laws and regulations.

2. Data Transmission Security

All data transmissions are encrypted to ensure security throughout the transfer process.

3. Sensitive Data Recommendations

Important: For AWR reports containing sensitive data, BIC strongly recommends that you:

  • Use the commercial edition instead of the public cloud version;
  • Mask or anonymize sensitive information before submission;
  • Upload only the data required for analysis and avoid excessive sensitive content.

4. Data Storage

The system automatically deletes uploaded AWR report files once the analysis is complete. Analysis results (excluding the original report) may be retained to improve services, but they will not contain any information that can identify individual users.

5. User Responsibility

Users must ensure that the uploaded AWR reports do not violate laws, regulations, or infringe upon the rights of others. Users bear full responsibility for the content they submit.

We are committed to safeguarding your data, but we encourage you to assess data sensitivity before use and select a more secure service option when necessary.

If you need further assistance, please contact the BIC support team.
